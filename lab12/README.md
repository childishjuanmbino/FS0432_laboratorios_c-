En este laboratorio se codifico la ecuacion de calor de una placa en funcion del tiempo y la posicion. Se mide la temperatura en todos los puntos en un tiempo t y se genera una snapshot, se repite esto hasta un cierto numero de pasos.

Para correr el programa solo hace falta descargar el archivo heat_2D y ejecutarlo usando el comando "python heat_2d.py". El propio programa creara su propio gif.
